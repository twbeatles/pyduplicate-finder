# 🔍 PyDuplicate Finder Pro

**PyDuplicate Finder Pro**는 파이썬(PySide6) 기반의 고성능 중복 파일 관리 도구입니다. 최신 멀티스레딩 기술과 스마트 캐싱을 통해 대용량 파일 시스템에서도 빠르고 정확하게 중복 파일을 탐색하며, 안전한 삭제 복구(Undo) 기능을 제공합니다.

---

## ✨ 주요 기능 (Key Features)

### 🚀 압도적인 성능 (High Performance)
- **초고속 스캔 (Fast I/O)**: `os.scandir` 기반의 재귀 스캔 엔진을 도입하여 기존 `os.walk` 대비 파일 탐색 속도를 비약적으로 향상시켰습니다.
- **최적화된 해싱 (BLAKE2b)**: 64비트 시스템에 최적화된 `BLAKE2b` 알고리즘을 사용하여 MD5보다 빠르고 안전하게 파일을 분석합니다.
- **스마트 캐싱 & 배치 처리**: `SQLite WAL` 모드와 대용량 배치(Batch) 처리를 통해 수십만 개의 파일도 끊김 없이 처리합니다.
- **부드러운 UI**: 결과 목록을 점진적으로 렌더링(Incremental Rendering)하여 대량의 결과 표시 중에도 앱이 멈추지 않습니다.

### 🛡️ 안전 및 정밀성 (Safety & Precision)
- **시스템 보호 (System Protection)**: Windows/Linux의 주요 시스템 폴더(Windows, Program Files 등)를 자동으로 건너뛰어 오작동을 방지합니다.
- **물리적 중복 방지 (Inode Check)**: 심볼릭 링크나 바로가기 등으로 인해 동일한 물리적 파일이 중복 집계되는 것을 원천 차단합니다.
- **안전한 삭제 & 실행 취소 (Robust Undo)**: 비동기 방식의 '삭제/복구' 작업을 지원하여, 대량의 파일을 삭제할 때도 UI가 멈추지 않습니다.
- **바이트 단위 비교**: 해시 충돌 가능성을 0%로 만들기 위한 바이트 단위 정밀 비교 옵션을 제공합니다.

### 🎨 모던 UI & 사용자 경험
- **직관적인 디자인**: 최신 트렌드의 플랫 디자인, 다크 모드/라이트 모드 지원.
- **편리한 사용성**: 우클릭 컨텍스트 메뉴(열기, 폴더 열기, 경로 복사), 드래그 앤 드롭 지원.
- **빈 폴더 정리기**: 비어있는 폴더를 재귀적으로 찾아내어 깔끔하게 정리해줍니다.
- **결과 내보내기**: 탐색된 중복 파일 목록을 CSV 파일로 저장하여 엑셀에서 관리할 수 있습니다.
- **미리보기**: 이미지, 텍스트, 코드 파일을 앱 내에서 즉시 미리보기(Monospace 폰트 적용) 할 수 있습니다.

## 📥 설치 방법 (Installation)

### 전제 조건
- Python 3.8 이상

### 설치 단계

1. **리포지토리 복제**
   ```bash
   git clone https://github.com/your-username/PyDuplicateFinder.git
   cd PyDuplicateFinder
   ```

2. **가상 환경 생성 (권장)**
   - Windows:
     ```bash
     python -m venv venv
     venv\Scripts\activate
     ```
   - macOS/Linux:
     ```bash
     python3 -m venv venv
     source venv/bin/activate
     ```

3. **의존성 설치**
   ```bash
   pip install -r requirements.txt
   ```

## 📖 사용 방법 (Usage Guide)

### 1. 프로그램 실행
```bash
python main.py
```

### 2. 검색 설정
- **파일 위치 추가**: '폴더 추가' 혹은 드래그 앤 드롭으로 검색할 위치를 등록합니다.
- **필터 옵션**:
    - `확장자`: 특정 확장자만 검색 (예: `jpg, png, mp4`). 비워두면 모든 파일 검색.
    - `최소 크기`: 너무 작은 파일은 무시하도록 설정 가능.
    - `시스템 폴더 보호`: 기본값으로 켜져 있으며, 끄지 않는 것을 권장합니다.

### 3. 스캔 및 중복 확인
- **스캔 시작**: 버튼을 누르면 빠른 속도로 스캔이 진행됩니다.
- **결과 확인**:
    - 왼쪽 트리 뷰에서 중복 파일 그룹을 확인할 수 있습니다.
    - 파일을 클릭하면 오른쪽 패널에서 **미리보기**가 표시됩니다.

### 4. 정리 및 삭제
- **자동 선택 (스마트)**: 각 그룹에서 '가장 최근에 수정된 파일' 혹은 '가장 오래된 파일' 등을 기준으로 남길 파일을 자동 선택합니다.
- **선택 항목 삭제**: 선택된 중복 파일들을 삭제(임시 이동)합니다.
- **실행 취소 (Undo)**: 삭제가 잘못되었다면 툴바의 `Undo` 버튼(Ctrl+Z)을 눌러 즉시 복구하세요.

### 5. 빈 폴더 찾기
- 툴바의 `빈 폴더 찾기` 아이콘을 클릭하면, 현재 추가된 검색 경로 내의 빈 폴더들을 찾아 일괄 삭제할 수 있습니다.

## 🏗️ 패키징 (Exe 파일 생성)
단독 실행 가능한 `.exe` 파일을 생성하려면 다음 명령어를 실행하세요:
```bash
pyinstaller PyDuplicateFinder.spec
```
생성된 파일은 `dist/PyDuplicateFinderPro.exe` 위치에 저장됩니다.
